## 🚀 快速开始

### 安装依赖
```bash
cd frontend
npm install
```

### 开发
```bash
npm run dev
```

### 构建
```bash
npm run build
```

## 📋 注意事项

1. **后端服务**：确保后端服务已启动（默认端口 8080）
2. **代理配置**：前端通过 Vite 代理访问后端 API
3. **Token 管理**：登录后 token 存储在 localStorage
4. **路由守卫**：后台页面需要登录才能访问
