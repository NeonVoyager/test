## 📁 项目结构

```
oneblog/
├── src/                    # 后端代码
│   ├── main/java/com/oneblog/
│   │   ├── controller/    # 控制器（前台+后台）
│   │   ├── service/       # 服务层
│   │   ├── mapper/        # 数据访问层
│   │   ├── entity/        # 实体类（13个）
│   │   ├── config/        # 配置类
│   │   ├── shiro/         # Shiro 配置
│   │   └── util/          # 工具类
│   └── main/resources/
│       ├── mapper/        # MyBatis XML
│       ├── sql/           # SQL 脚本
│       └── application.yml
│
└── frontend/               # 前端代码
    ├── src/
    │   ├── api/           # API 接口
    │   ├── components/    # 组件
    │   ├── layouts/       # 布局
    │   ├── router/        # 路由
    │   ├── stores/        # 状态管理
    │   ├── utils/         # 工具函数
    │   └── views/         # 页面
    ├── package.json
    └── vite.config.js
```

## 🚀 快速开始

### 1. 后端启动

```bash
# 配置数据库（修改 application.yml）
# 执行 SQL 脚本
mysql -u root -p oneblog < src/main/resources/sql/init.sql

# 启动后端
mvn spring-boot:run
```

### 2. 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 3. 访问地址

- 前台：http://localhost:3000
- 后台：http://localhost:3000/admin/login
- 后端API：http://localhost:8080/api
